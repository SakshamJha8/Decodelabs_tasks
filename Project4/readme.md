# Project 4: NLP & Sentiment Analysis

# Overview
This project focuses on Natural Language Processing (NLP) to classify product reviews as Positive or Negative. It demonstrates the complete text classification workflow, from preprocessing raw text to building and evaluating a machine learning model.

# Objective
Develop a sentiment analysis model capable of understanding unstructured text and predicting the sentiment of customer reviews.

# Project Workflow
- Load and explore the dataset
- Perform text preprocessing:
  - Lowercasing
  - Tokenization
  - Stopword removal
  - Lemmatization
- Convert text into numerical features using TF-IDF Vectorization
- Train a machine learning classifier (Naive Bayes or SVM)
- Evaluate model performance using standard classification metrics

# Technologies Used
- Python
- Pandas
- NumPy
- NLTK / spaCy
- Scikit-learn
- Matplotlib
- Seaborn

# Machine Learning
- Feature Extraction: TF-IDF Vectorizer
- Classification Model: Naive Bayes or Support Vector Machine (SVM)

# Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

Built an end-to-end NLP pipeline capable of preprocessing text, extracting meaningful features, and accurately classifying customer reviews into positive or negative sentiments.
